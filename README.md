# Restaurant Kitchen Service

## Overview
Django-based web application for managing restaurant kitchen workflow.  
Includes dashboard, dishes management, and cooks management.

## Features
- Dashboard with navigation
- Dish list and detail pages
- Cook list and detail pages
- Custom templates with Material Kit integration
- Footer with logo, navigation, and contacts
